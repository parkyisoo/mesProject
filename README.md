MES-Lite

제조 현장 운영 흐름을 직접 설계하고 구현한 경량 MES 포트폴리오를 만들었다.

기술스택
Java 17, Spring Boot 3.2.3, Thymeleaf, JPA, MySQ

개발기간
약 2주

구현 기능
로그인과 역할 기반 접근 제어(ADMIN / OPERATOR / QC) 구현
작업지시를 생성하고 상태를 관리하게 만듦
LOT를 State Machine 방식으로 추적하고 이력을 자동 기록하게 만듦
생산실적 입력 기능을 만들었고, 수량 검증과 LOT 자동 완료 처리
불량 관리 기능 구현, 원인 분류와 처리 방법을 기록하게 했음
KPI 리포트, 수율, 불량률, UPH를 Chart.js 차트로 구현
대시보드를 만들고, 현황 요약과 불량 경고를 넣었음.
